Alpha commands.

The first release focuses on the safety layer: setup, backup, and health checks.

aicodebackup setup Create the safety layer

Initializes Git, connects GitHub, creates a private repo, and runs the first backup.

aicodebackup backup Save the current work

Adds, commits, and pushes changes without requiring the user to understand Git.

aicodebackup doctor Check backup health

Checks Git, GitHub CLI, auth, remote, working tree, and the latest backup.

Built for Windows and macOS.

AICodeBackup is designed around the onboarding paths new AI coding users are most likely to hit first.

Windows PowerShell

Setup attempts Git and GitHub CLI installation through winget when tools are missing.

Apple macOS Terminal

Setup uses Homebrew when available and starts browser-based GitHub login.

Does AICodeBackup create private repositories?

Yes. The setup flow creates GitHub repositories as private by default.

Do I need to know Git?

No. The alpha wraps the setup, commit, and push workflow behind simple commands.

Is this production-ready?

AICodeBackup is currently an alpha release for small-group testing and feedback.
