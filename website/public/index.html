Commands that do the heavy lifting.

Three commands. That's it.

aicodebackup setup

Initial setup. Installs requirements, connects GitHub, and creates your private repository.

aicodebackup backup

Save your current project. Commits, and pushes changes to GitHub.

aicodebackup doctor

Check backup health, repo status, authentication, and more.

Frequently asked questions.

Do I need to know Git?

No. AICodeBackup wraps Git setup, commits, and pushes behind simple commands.

Is my code really private?

Yes. The setup flow creates GitHub repositories as private by default.

Where is my code stored?

Your code is stored in your own private GitHub repository.

What happens if I uninstall?

Your GitHub repository and backups stay in your GitHub account.

Is AICodeBackup free?

The alpha CLI is free and open source under the MIT License.
