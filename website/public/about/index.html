About AICodeBackup.

AICodeBackup is an open-source CLI for people building software with AI coding tools who need a simple private GitHub backup habit before local work disappears.

Why this project exists

AI coding tools help users create and change code quickly, but local projects can still be lost through resets, accidental deletion, failed experiments, or unclear file copies. AICodeBackup gives those projects a safer path to private GitHub repositories without requiring the user to learn Git before the first backup.

What AICodeBackup does

The CLI checks local requirements, helps connect GitHub, creates or connects a private repository, and runs backups from the terminal. It focuses on practical recovery points: setup, backup, doctor, and watch commands that make it easier to know whether current AI-generated code is protected.

Project status

AICodeBackup is currently alpha software and is published as an open-source package under the MIT License. The source code is available on GitHub, and the installable package is available on npm.

Maintainer and support

The public project is maintained through the AICodeBackup GitHub repository. For support, privacy questions, or project feedback, contact hello@aicodebackup.com.
