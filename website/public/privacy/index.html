Privacy Policy.

Last updated: June 17, 2026.

Overview

AICodeBackup is an open-source alpha CLI and website. This policy explains the basic privacy expectations for the public website and the CLI.

Website information

The website is a static site hosted on Vercel. Hosting providers may process standard request information such as IP address, user agent, requested URL, and timestamps for security, delivery, and operational purposes.

Website analytics

The website uses Google Analytics to understand aggregate traffic and product-interest events, such as page views, install command copies, npm, GitHub, and email clicks, primary call-to-action clicks, and language switching. These analytics do not collect source code, local project files, GitHub repository contents, or CLI backup data.

CLI behavior

The alpha CLI runs on your machine and uses local Git, GitHub CLI, and GitHub when you choose to set up or run backups. Repository creation, authentication, and pushing code are handled through GitHub tools and GitHub services.

Contact

For privacy questions, contact hello@aicodebackup.com.
