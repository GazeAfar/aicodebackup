别让 AI 写出来的代码丢失。

AICodeBackup 会把你的 AI 编程项目备份到 GitHub 私有仓库。你不需要先学会 Git，也不需要记住复杂命令。

npm install -g aicodebackup@alpha
安装 alpha 查看 GitHub
  • 默认创建私有仓库
  • 支持 Windows 和 macOS

AI 写代码很快。丢代码也很快。

误删或崩溃

电脑故障、误操作或 AI 改坏文件，都可能让本地项目不安全。

没有远程备份

AI 工具负责写代码，但通常不会替你把代码安全保存到 GitHub。

版本混乱

手动复制文件夹会很快变乱，不知道哪一份才是最新可用版本。

隐私风险

公开仓库或不合适的云同步，可能让未发布代码暴露出去。

三步完成安全备份。

初始化一次，之后重复备份。

创建安全层

检查 Git 和 GitHub CLI，连接 GitHub，并创建 GitHub 私有仓库。

保存当前工作

AICodeBackup 会自动 add、commit 和 push，不要求你理解 Git 细节。

默认私有备份

项目会备份到你的 GitHub 私有仓库，避免源代码被意外公开。

三个核心命令。

第一版只解决最重要的备份问题。

aicodebackup setup

初始化项目，安装或检查依赖，连接 GitHub，并创建私有仓库。

aicodebackup backup

把当前项目状态提交并推送到 GitHub，形成一个安全恢复点。

aicodebackup doctor

检查 Git、GitHub CLI、登录状态、远程仓库和最近备份状态。

支持 Windows 和 macOS。

面向 AI 编程用户常见的本地开发环境。

Windows PowerShell

当缺少 Git 或 GitHub CLI 时，setup 会尝试通过 winget 引导安装。

macOS Terminal

setup 会在可用时使用 Homebrew，并通过浏览器完成 GitHub 登录。

常见问题。

我需要会 Git 吗？

不需要。AICodeBackup 把常见 Git 操作包装成 setup、backup 和 doctor 命令。

仓库是私有的吗？

是的。setup 流程默认创建 GitHub private repository。

我的代码保存在哪里？

代码保存在你自己的 GitHub 私有仓库里，不是保存在 AICodeBackup 的服务器上。

卸载 AICodeBackup 会影响备份吗？

不会。已经推送到 GitHub 的仓库和历史备份仍然保留在你的 GitHub 账号中。

AICodeBackup 免费吗？

当前 alpha CLI 是免费的开源版本，使用 MIT License。
